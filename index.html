Tentang Saya

Saya adalah seorang siswa SMK di Karawang yang memiliki minat dalam bidang IT.

Saya bersekolah di SMK Negeri 1 Karawang jurusan Rekayasa Perangkat Lunak, saya berusia 17 tahun dan lahir di Jakarta.

Selain belajar teknologi, saya juga memiliki hobi membaca buku, mendengarkan musik, dan bermain olahraga seperti bulutangkis.

Pengalaman

  • PKL di perusahaan IT lokal, membantu pengembangan aplikasi berbasis web.
  • Mengerjakan beberapa proyek kecil seperti pembuatan website portofolio dan sistem manajemen sederhana.
  • Mengikuti berbagai lomba IT tingkat sekolah dan regional, meskipun belum berhasil menjadi juara, saya terus berusaha belajar dari pengalaman.

Visi Masa Depan

Saya bercita-cita menjadi seorang Software Engineer yang mampu menciptakan solusi teknologi inovatif dan bermanfaat bagi masyarakat.

Saya juga berharap dapat melanjutkan pendidikan di bidang IT untuk memperdalam ilmu saya.
